# Animations

```css
@keyframes A {
	0% {
		color: green;
	}
	100% {
		color: cornflowerblue;
	}
}


@keyframes B {
	from {
		color: green;
	}
	to {
		color: cornflowerblue;
	}
}


h2 {
	font-weight: bold;
	font-size: 3em;
	animation: A 3s;
	animation-delay: 2s;
	animation-iteration-count: infinite;
}

```
