# Proxy
* If "c" is not present in a proxy, 37 is returned

```javascript
const manager = {
    get(source, prop) {
        if(prop in source){
            return source[prop];
        }else{
            return 37;
        }
    }
};

const p = new Proxy({}, manager);
p.a = 1;
p.b = undefined;

console.log(p.a, p.b); // 1, undefined
console.log('c' in p, p.c); // false, 37
```
# Error handling 

```javascript
var a = 100; 
var b = 0; 
try { 
   if (b == 0 ) { 
      throw(“Divide by zero error.”); 
   } else { 
      var c = a / b; 
   } 
} 
catch( e ) { 
   console.log("Error: " + e ); 
}
```
